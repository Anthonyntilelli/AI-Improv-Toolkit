#  Actions
1) Make example config files.
2) Read in config files.
3) Validate config files. (pydantic-settings?)
4) Connect to the NATS server.
5) Set status to "starting".
6) Load setting into the key-value store.
7) Set status to "ready".
8) Subscribe to relevant NATS topics (interface) and listen for reset.

# Reset Actions
1) Set status to "resetting".
2) Clear key-value store.
3) Reload config files.
4) Run "Actions" from top.
